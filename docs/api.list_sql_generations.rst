List SQL generations
============================

request this ``GET`` endpoint to get a list of all SQL generations

    /api/v1/sql_generations


**Parameters**

.. csv-table::
   :header: "Name", "Type", "Description"
   :widths: 20, 20, 60

   "prompt_id", "string", "the prompt id, ``Optional``"


**Responses**

HTTP 200 code response

.. code-block:: rst

    [
        {
            "id": "string",
            "metadata": {},
            "created_at": "string",
            "prompt_id": "string",
            "finetuning_id": "string",
            "status": "string",
            "completed_at": "string",
            "sql": "string",
            "tokens_used": 0,
            "confidence_score": 0,
            "error": "string"
        }
    ]

**Request example**

.. code-block:: rst

   curl -X 'GET' \
  'http://localhost/api/v1/sql-generations?prompt_id=6595ad38fef5f74dd00695a3' \
  -H 'accept: application/json'
